# Hello World Django App

This is a simple Django application that provides:
- A JSON response with the message `{"Message": "Hello World!"}`.
- (Optional) An HTML page displaying "Hello World!" in bold.

---

## Features
1. **JSON Endpoint**: Returns a "Hello World!" message in JSON format.
2. **Optional HTML Endpoint**: Displays "Hello World!" in an HTML page.

---

## Prerequisites
- Python (version 3.8+)
- pip (Python package installer)
- Django (version 5.1 or later)

---
